Project screenshots
===================

Current files are SVG cover images (flixvault.svg, ecommerce.svg, etc.).

To use real screenshots from your projects:
1. Take a screenshot (1200×630 or 16:9 recommended)
2. Save as PNG with the same name, e.g. flixvault.png
3. Update src in src/data/projectsData.ts:
   image: '/images/projects/flixvault.png'

4. Update github and liveDemo URLs in projectsData.ts when ready.
